# noslop checklists

Fill for the current piece. Do not ship without PRE-WRITE, honest features, and SCORE evidence.

## PRE-WRITE

```text
NOSLOP PRE-WRITE
Audience:
Length / form:
Specific anchors (name / number / place / time):
Aftermath plan (what happens after climax — required):
End turn / twist (what recontextualizes the piece):
Embodied emotion beat (body + sensation):
Frame device if any (log / later tell / none):
Theme surface (where meaning is allowed to show — not silent):
Rhythm note:
Surface risk:
```

Rules:

- Anchors must be concrete.
- Aftermath and end turn are required for fiction; nonfiction maps to consequence + reframe.
- Theme surface: name the beat where meaning shows (not “none”).

## POST-WRITE structure grade

```text
NOSLOP GRADE
Theme:       SCORE 0-2  PASS|FAIL|N/A  — one line
Plot:        SCORE 0-2  PASS|FAIL|N/A  — one line
Time:        SCORE 0-2  PASS|FAIL|N/A  — one line
Specificity: SCORE 0-2  PASS|FAIL|N/A  — one line
Felt life:   SCORE 0-2  PASS|FAIL|N/A  — one line
MEAN: x.x
MERGED: PASS|FAIL
FIX: …
```

| Axis | SCORE 2 | SCORE 0 |
|------|---------|---------|
| Theme | Meaning surfaces in scene without pure TED close | Silent theme **or** sermon-only |
| Plot | Aftermath + turn present | Hard cut; no consequence |
| Time | Anchors / order serve the piece | Fog only |
| Specificity | Real anchors land | Abstract fog |
| Felt life | Embodied sensation | Label-only emotion |

Merge: mean of scored axes ≥ 1.2 and ≤1 FAIL → PASS.
