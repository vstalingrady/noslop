# noslop checklists

Fill these for the current piece. Do not ship without both filled.

## PRE-WRITE (create for this task)

```text
NOSLOP PRE-WRITE
Audience:
Length / form:
Specific anchors I will use (name / number / place / time — at least one):
One deliberate mess, grey choice, or open thread:
How theme will be shown (not stated):
Rhythm note (where short hits land):
Surface risk for this genre (one line):
```

Rules for filling:

- Anchors must be concrete, not “something specific later.”
- Mess/grey/open thread: skip only if form is too short (e.g. one-line reply); write `N/A — too short`.
- Theme line: describe the *scene or detail* that carries meaning, not the moral sentence.

## POST-WRITE grade

```text
NOSLOP GRADE
Theme:       SCORE 0-2  PASS|FAIL|N/A  — one line, cite a span if FAIL
Plot:        SCORE 0-2  PASS|FAIL|N/A  — one line
Time:        SCORE 0-2  PASS|FAIL|N/A  — one line (short text → N/A)
Specificity: SCORE 0-2  PASS|FAIL|N/A  — one line
Felt life:   SCORE 0-2  PASS|FAIL|N/A  — one line
MEAN: x.x   (ignore N/A)
MERGED: PASS|FAIL
FIX: (if FAIL) structural bullets only
```

### Axis guide

| Axis | SCORE 2 | SCORE 0 |
|------|---------|---------|
| Theme | Meaning stays in action/image | Closing sermon or stated moral |
| Plot | Friction or open thread fits the piece | Everything bows neat; restatement closers |
| Time | Order serves the piece (jump/aside OK) | Long piece marches only linear with no break |
| Specificity | Real anchors land | Abstract fog only |
| Felt life | Plain feeling or human mess | Body-catalog only / no doubt |

### Merge

- Drop N/A axes.
- **PASS** if mean of scored axes ≥ 1.2 and at most one FAIL.
- **FAIL** otherwise.
- On FAIL: apply FIX, rewrite, re-fill GRADE. Max 2 full grade rounds.

## Surface polish

After structure PASS (or final pass), scan [style-and-bans.md](style-and-bans.md) once.  
Do not start with bans — structure checklists first.
