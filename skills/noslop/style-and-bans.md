## Hard bans (don’t use)

### Words

delve / delving, tapestry, realm (figurative), landscape (figurative industry fluff), testament (“a testament to”), vibrant, pivotal, intricate / intricacies, meticulous / meticulously, bolster, garner, underscore, multifaceted, nuanced (as empty filler), foster / fostering, leverage (as verb fluff), utilize (say **use**), commence (say **start**), facilitate, encompass, paramount, groundbreaking, cutting-edge, game-changer / game-changing, transformative, revolutionize, seamless / seamlessly, robust (outside real eng meaning), comprehensive (self-praise), endeavor, aforementioned, harnessing, spearheading, showcasing, highlighting, emphasizing, enhancing, unprecedented, remarkable, stunning, profound (empty), synergy, pain points, value-add, moving forward, touch base, circle back, rest assured, it goes without saying, holistic, paradigm, unlock / unleash (metaphor spam), empower, streamline, elevate, navigate (figurative complexities)

## Soft bans (rare / only if exact)

- robust, seamless, scalable, innovative — OK in real eng/product specs, not as vibes
- landscape — OK for actual land/scenery
- vital, crucial — OK once if true; not every sentence
- leverage — OK in finance “leverage” sense, not “leverage AI”

## Style rules (surface)

**Rhythm**
- No three same-length sentences in a row.
- Don’t chain short. Punchy. Fragments. As a whole paragraph (parataxis spam).
- Mix 4-word hits with longer ones.

**Lists**
- Don’t default to rule of three. Two, one, four — fine.

**Punctuation**
- Em dash: max ~1 per 500 words. Prefer commas, periods, parens.
- Exclamation: almost never.
- Ellipsis: only real trailing-off, not transitions.

**Tone**
- Contractions OK: don’t, can’t, it’s.
- Pick a side. One hedge max per claim when unsure — no hedge chains.
- Active voice. Kill “is being done / was found to be.”
- No corporate pep talk. Include friction when real.

**Formatting (context)**
- Emails/DMs: no markdown headers, no bold spam, no emoji bullets.
- Social: no hashtag stacks, no “Thread:” cosplay unless user wants it.

**Honesty**
- No invented stats, quotes, or “studies show” without a real source.
- Hypotheticals marked as hypotheticals.

## Swap cheatsheet

| Avoid | Prefer |
|-------|--------|
| delve into | look at, dig into, check |
| utilize | use |
| leverage | use, draw on |
| facilitate | help, make easier |
| robust | solid, reliable (or name the property) |
| seamless | smooth, no extra steps |
| landscape | field, industry, space (or be specific) |
| tapestry | mix, mess, set of |
| pivotal | key, main, the thing that broke |
| underscore | show, point out |
| furthermore / moreover | also, and, but |
| it’s worth noting | just say the thing |
