# Style + banned surface tells

**Required ship pillar** with construction ([construction.md](construction.md) · `SKILL.md`).  
Structure without this pass still reads like template AI. This pass without structure still reads like AI fiction.

Surface hygiene is **not** optional and **not** scoremax — floor on words, phrases, rhythm, punctuation, formatting.  
Long fiction: structure first, then this file hard. Short prose: this file is most of the skill.

Lists draw on common AI-tell research and open skill packs (see [THIRD_PARTY_NOTICES.md](../../THIRD_PARTY_NOTICES.md)); reorganized for noslop and paired with StoryScope construction, which those packs do not cover.

## Hard bans — openers (sentence / reply start)

- Certainly, / Absolutely, / Sure, / Great question! / That’s a great point!
- I’d be happy to…
- Moreover, / Furthermore, / Additionally, / Notably, / Importantly, / Interestingly, / Indeed, (empty openers)

**First-word smell (don’t open a whole reply with these on autopilot):** yes / sure / certainly / here / based / creating / title / step / introduction / comprehensive — vary the start; just say the thing.

## Surface structure (how AI stacks clean words)

These show up even when vocabulary is clean. Kill them.

| Rule | Do | Don’t |
|------|-----|--------|
| **Length mix** | Mix ~4-word hits with longer lines | Three same-length sentences in a row |
| **No parataxis spam** | Connect with because / but / which / ; | Short. Short. Short. as a whole para |
| **No rule of three** | Two, one, four, five | Default triad of everything |
| **No hedge seesaw** | Pick a side; one counter max | Equal weight both ways forever |
| **No pep talk** | Friction, doubt, mess when real | Cheerleading / “rewarding journey” |
| **Paragraph shape** | Uneven; some one-liners; some abrupt ends | Topic → explain → example → transition every time |
| **Bullets** | Sparse; uneven length; ≤5–7 | Bullet essay; ✅ emoji stacks |
| **Active** | Subject does the thing | “is being done / was found to be / are considered” |
| **Ugly OK** | Fragment. Run-on when the thought isn’t done | Every line polished the same way |
| **Specific** | Names, numbers, times, places | “various networks,” “significant growth” |
| **Less obvious word** | Second choice sometimes | Always highest-probability token |

**Punctuation**

- Em dash: **max ~1 per 500 words** (biggest surface tell). Prefer comma, period, paren, semicolon, colon.
- Exclamation: ~1 per 1,000 words max; almost never in long fiction.
- Ellipsis: real trailing-off only; not as transition.
- Semicolons / colons: fine when they earn their keep.

**Formatting by context**

- Emails / DMs / SMS: no markdown headers, no bold spam, no asterisk cosplay.
- Social: no hashtag stacks; no “Thread:” / 🧵 unless user asked; emoji 0–2, not line leaders.
- Reports / docs: headers OK when the form needs them.

**Honesty**

- No invented stats, studies, quotes, or fake anecdotes.
- Hypotheticals marked (*imagine…* / *suppose…*). Fabricated specificity is worse than honest vagueness.
- Real names and dates when you have them.

**Voice**

- Contractions: don’t, can’t, it’s.
- Match the person if known (slang, length, humor, what they would never say).
- Default: direct, slightly informal, may start with And/But, trusts the reader.

## Swap cheatsheet

| Avoid | Prefer |
|-------|--------|
| delve into | look at, dig into, check |
| utilize | use |
| leverage (fluff) | use, draw on |
| facilitate | help, make easier |
| robust (vibe) | solid, reliable — or name the property |
| seamless | smooth, no extra steps |
| landscape (fluff) | field, industry — or name it |
| tapestry | mix, mess, set of |
| pivotal | main, the thing that broke |
| underscore | show, point out |
| furthermore / moreover | also, and, but |
| it’s worth noting | just say the thing |
| empower / elevate / unlock | say what actually changes |
| comprehensive | drop it, or list what is covered |

**Instead of vibes**

| Weak | Stronger |
|------|----------|
| powerful analytics | paste the address; it says you run out of cash in 47 days |
| seamless UX | three clicks from connect to first score |
| significant growth | 34 users week one; 12 came back day two |
| various chains | Solana, specifically |
| rewarding journey | RPC died at 3am; almost killed the feature |

## Pre-ship surface check (silent — don’t narrate)

1. Any hard-ban word / phrase / opener? → cut or rewrite  
2. Three same-length sentences in a row? → break  
3. Parataxis spam (short. short. short.)? → connect or merge  
4. Everything in threes? → break  
5. Hedging seesaw? → pick a side  
6. Em dash over budget? → remove extras  
7. Passive sludge? → active  
8. Every para ends with a transition? → cut some  
9. Invented specifics? → remove or mark hypothetical  
10. Could any model write this for any person? → add one real anchor  
11. Still smells like ChatGPT? → rewrite until no  

Apply silently. Never say “as per the skill” or “per the guidelines.”
