# Story construction (long fiction)

Paper: [arXiv:2604.03136](https://arxiv.org/abs/2604.03136).  
**Structure first. Sparse human moves. Not a score recipe.**

Ship bar = a careful reader finishes the page without template nausea.  
VOICE / StoryScope / P(human) = footnotes only — never the win.

## Length budget

| Length | Construction budget |
|--------|---------------------|
| &lt; ~400 words | Almost **no** novel structure. Anchors + one concrete beat. |
| ~400–1.5k | Pick **ONE**: grey choice **or** one time warp **or** open cost. |
| ~1.5k–5k+ | Full construction space — still **sparse** (2–4 moves). |

## PRE-STRUCTURE (required before prose on long fiction)

```text
STORY PRE-STRUCTURE
Premise:        (ordinary human stakes — not the tool/repo/scores)
POV / distance:
Moral grain:    what choice is greyer than tidy? open cost?
Time shape:     linear | one jump | frame | braided  (only if length earns it)
Tracks:         one spine + optional B-track / loose end
Theme location: scene/action only — never end TED
World anchors:  2–4 named concrete things
Emotion policy: mix named feeling + body; ban body-every-paragraph
Ending type:    acceptance-bow banned unless earned; prefer open / cut / mixed agency
Diversity seed: one human-leaning move I will NOT use this draft
Anti-AI defaults refused: theme dump | single-track tidy | setting-as-mood spam
Palette picks (2–4 max): …
```

No PRE-STRUCTURE on long fiction → do not draft.

## Sparse palette (pick 2–4 only)

1. Grey / mixed agency at the hinge  
2. One real temporal discontinuity **if length allows**  
3. One subplot or unresolved side thread  
4. Named outside world (book, brand, street, song — specific)  
5. End without moral restatement (image / action / cutoff / open cost)  
6. Occasional direct feeling word *or* reader aside — not body-formula soup  
7. Dialogue that advances relationship/plot, not seminar ethics  

**Stacking all seven = FAIL** (new model cluster).  
**Diversity seed** forces at least one unused move so same-brief drafts are not twins.

## Draft for a reader

- Scenes carry theme; narrator does not close with “what we learned.”  
- One boring true detail that doesn’t “earn” its place is fine.  
- Incomplete beats OK; don’t pay off every setup.  
- Resolution: mixed agency / unfinished cost OK; “she finally understood” as last beat is the AI default.

## Structural FIX (max 2 rounds)

| Symptom | Fix (structure) | Not a fix |
|---------|-----------------|-----------|
| Lesson in last para | Cut lesson; end on image/action | Softer synonyms |
| Too tidy | Break one causal link; leave a cost open | More adjectives |
| Flat time (long form) | Move one scene out of order; delay a fact | Random “meanwhile” |
| Body/sense spam | Name the feeling once; cut weather-as-mood | New sensory list |
| Vague world | Swap one allusion for a real name | “Rich detail” filler |

FIX = synonym swaps only → **FAIL**.

## After structure (style is still required)

Construction does **not** replace surface hygiene. After structural FIX:

1. **Full style + bans pass** — required ([style-and-bans.md](style-and-bans.md)):
   - Hard-ban words/phrases: **zero** hits  
   - Soft bans only in literal/technical sense  
   - Rhythm, em-dash budget (~1/500 words), no template openers  
   - Swap cheatsheet for leftover glue  
2. VOICE hard fails only (sermon / ban spam / zero anchors on long text) — anti-glue floor, not “higher score = more human.”  
3. Optional StoryScope lab with honest span cites — never forge; never sole ship gate.  
4. Ship when a careful human would finish the page **and** the ban list is clean.

Skipping bans because “the plot is grey enough” = FAIL.

## Short agent prose

**Do not use this file** for email, bio, blurb, short answers.  
Flow + anti-glue + real anchors only. No novel toys (aftermath, frame/memoir, theme stacks).
